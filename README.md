# CryptoDashboard
Streamlit-based dashboard that pulls public market data (Bybit) and computes breakout / residual momentum signals.

# Intructions:
1. Download the cryptodashboard.py file, and put it in a location of your choice.
2. Download the runner.txt file and put it in the same location as the above.
3. Open the runner.txt file and put in between the " " the path that the cryptodashboard.py is.
4. save the runner.txt as a .bat file, by renaming it to runner.bat when saving.
5. Execute the runner.bat file

Be aware that the script takes a while to load.

This Script was vibe coded, and as such there may be errors, if someone wants to improve/ give suggestions, just make a pull request or contact me on twitter: @lumocrypto
